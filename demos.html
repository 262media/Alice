Alice Demos

Alice (A Lightweight Independent CSS Engine) is a micro JavaScript library focused on using hardware-accelerated capabilities (in particular CSS3 features) in modern browsers for generating high-quality, high-end visual effects.

Demos

Bounce

Bounce - The "bounce" effect simulates the bouncing of an object from a top view including a shadow.

alice.bounce({
    id: "my_bounce"
});
Toss

Toss - The "toss" effect is similar to throwing an object like a frisbee. With the options, you can specify the direction that the toss comes from.

alice.toss({
    id: "my_toss",
    duration: 2000, // ms
    iteration: 'infinite'
});
Wobble

Wobble - The "wobble" effect simulates the behavior you get when you press-and-hold on icons to delete apps. However, the name can be a misnomer because by adjusting a few options, you can get a "breeze" or "sway" effect.

alice.wobble({
    id: "my_wobble"
});
Spring

Spring - The "spring" effect simulates the "springiness" of an object being dropped. It can be configured with a certain amount of elasticity, which affects the number of bounces and how "squishy" the object appears.

alice.spring({
    id: "my_spring",
    width: 96,
    height: 96,
    drop_height: 190,
    elasticity: 0.56, // basketball
    iteration: 'infinite'
});
Fade

Fade - The "fade" effect fades from one DIV element to another.

alice.fade2({
    id: "my_fade",
    duration: 3.0,
    iteration: 'infinite'
});

Rotate - The "rotate" effect rotates a DIV. By specifying a few parameters, rotate can simulate other effects like a pendulum, wobble, or swinging door.

alice.rotate("my_rotate", "45", {
    origin: "50% 0%",
    timing: 1000,
    iteration: 'infinite',
    direction: 'alternate',
    easing: 'cubic-bezier(0.33333,0.6667,0.66667,1)' // bounce
});

Cube

Cube - The "cube" effect takes DIV elements and places them on to the faces of a cube, which can be sliced into horizontal layers.

The cube can be rotated with touch, mouse, keyboard, and tilt events. With slices enabled, you get a very cool effect of each slice rotating one right after the other.

alice.cube({
    id: "my_cube",
    hslice: 8
});
