Alice

Alice - (A Lightweight Independent CSS Engine) - is a highly focused JavaScript/CSS3 library for generating high-quality, high-end visual effects.

View Demos

To download a version for the PlayBook:

  1. If you haven't installed the WebWorks SDK for Tablet OS, follow these instructions before continuing.
  2. Download alice.bar and save it to a temporary folder on your machine.
  3. Open a Command Prompt and run these commands (Note: you'll need to adjust the paths for your machine):
    1. cd "C:\Program Files\Research In Motion\BlackBerry WebWorks SDK for TabletOS 2.1.0.6\bbwp\blackberry-tablet-sdk\bin"
    2. blackberry-deploy -installApp -password your password -device your IP Address -package C:\your temporary folder\alice.bar
  4. An icon should appear at the bottom of your PlayBook screen. Tap it to launch Alice.

You can also download a zipped version.
